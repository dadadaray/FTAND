 FTAND is a database of forest food nutrient and taxonomy annotations built on detailed surveys of forest ecosystems and a comprehensive analysis of multiple data sources. It is a research-oriented dataset that provides information on the types, attributes, and nutritional values of forest foods. The data set contains a total of 163 entries, encompassing globally representative forest foods whose nutrient composition has been clarified in the current research field, for example, ginkgo, pine nuts, venison and royal jelly. Also, taxonomic information that is categorized from kingdom to species. In addition, nutritional content including proteins, fats, carbohydrates, vitamins, and minerals. FTAND is a useful tool to advance the research and sustainable use of forest foods and provides a solid data base to assess the nutritional and health benefits of forest foods worldwide. It aims to unlock the potential of forest foods to enhance dietary diversity and improve nutritional status, thereby contributing to global food security and sustainable development.
